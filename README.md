## Voraussetzungen

Für das Ausführen der Anwendung werden benötigt:

- Node.js
- npm
- Internetverbindung


## Installation und Start

Repository klonen:

git clone https://github.com/SgarrX/Creepets

cd Creepets

In den Frontend-Ordner wechseln:

cd frontend

Abhängigkeiten installieren:

npm install

## Anwendung starten

npm run dev

Die Anwendung ist danach erreichbar unter:

http://localhost:5173
